### Installation

1. `git clone https://github.com/flutter/flutter.git -b stable\n`
2. `flutter precache`
3. `flutter doctor`
4. `flutter doctor --android-licenses`
   - android
     1. install flutter plugin
     2. android simulator run
     3. build
  
   - ios
     1. `open -a Simulator`
     2. `flutter run`
